# Installation
## Clone repo
```
  git clone https://github.com/Manas140/paradise.git && cd paradise
```
## Install dark
<p align="center">
  <img src="./paradise-dark.png">
</p>
```
  ./install.sh d
```
## Install light
<p align="center">
  <img src="./paradise-light.png">
</p>
```
  ./install.sh l
```
